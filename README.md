# 🛒 ShopFlow Backend

Production-grade e-commerce REST API built with Node.js + Express.

## API Endpoints
- `POST /api/auth/register` — Register new user
- `POST /api/auth/login`    — Login (TODO)
- `GET  /api/products`      — List products
- `POST /api/payment/checkout` — Process payment

## Setup
```bash
npm install
npm start
```
